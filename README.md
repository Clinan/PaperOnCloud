# PaperOnCloud
## 功能说明
- 使用EasyAR进行目标识别，然后将模型展示出来。
- 现在支持展示带骨骼动画的模型，可以添加音乐。
---
## 待开发功能

- 网络下载场景，并自动设置识别加载
